FCodeBlock 组件 ​

概述 ​

代码块组件是一个用于显示代码的组件，提供了许多功能，如复制代码、代码显示隐藏动画、主题切换等。 由于为了减少体积，需要额外引入shiki库依赖。

shiki官方文档地址

sh
npm install -D shiki
sh
yarn add -D shiki
sh
pnpm add -D shiki
sh
bun add -D shiki

默认使用 ​

可以使用title、code、lang、showCode 和 theme 属性来定义代码块组件。

  • title属性定义代码块的标题。
  • code属性定义要显示的代码。
  • lang属性可以选择：js、html、ts、css，默认值是：js。
  • showCode属性可以选择：true、false，控制代码是否显示。
  • theme属性可以选择：light、dark，默认值是：light。
